<pitfalls>
P-01: The contacts list page URL is /contacts, but the page heading says "Clientes e Fornecedores". isLoaded() should match the URL path /contacts.
P-02: The delete button is a <Button> with title="Excluir" — use page.getByTitle("Excluir") scoped to the row. The row is identified by the contact name link.
P-03: The "Novo Cadastro" button is a <Link href="/contacts/new"> wrapped in a Button — use getByRole("link", { name: /novo cadastro/i }).
P-04: PhoneInput may mask input — fill with "(11) 99999-0001" format. Verify with page.inputValue() after fill.
P-05: After contact creation, the form redirects to /contacts. Wait for URL to match /contacts$ (exact) to confirm creation completed.
P-06: API-created contacts for the delete test must use a unique name per test run (use Date.now() suffix) to avoid collisions with concurrent test runs.
P-07: The DataTable renders contact rows where the name is a <Link> to /contacts/{id}. Row identification: locator("tr, [role='row']").filter({ has: page.getByRole("link", { name: contactName }) }).

### Task 1: Create ContactsPage POM (`e2e/pages/contacts.page.ts`)

Create a new file `e2e/pages/contacts.page.ts` with a class-based POM following the exact pattern of `e2e/pages/proposals.page.ts`.

The class must have:

**Constructor properties:**
- `page: Page`
- `newContactButton: Locator` — `page.getByRole("link", { name: /novo cadastro/i })`
- `pageHeading: Locator` — `page.locator('h1').first()`

**Methods:**

`goto()`: Navigate to `/contacts`.

`isLoaded()`: Wait for URL to match `/contacts$` (15s timeout), return true.

`getContactByName(name: string)`: Return `this.page.getByRole("link", { name }).first()` (contact names render as links to /contacts/{id}).

`createContact(data: { name: string; phone: string })`:
1. Click `newContactButton`
2. Wait for URL `/contacts/new` (15s)
3. Wait for name input `#name` or `input[name="name"]` to be visible (15s)
4. Fill name: `page.locator('input[name="name"], #name').fill(data.name)`
5. Fill phone: `page.locator('input[name="phone"], [data-testid="phone-input"] input, input[placeholder*="telefone" i], input[placeholder*="phone" i]').first().fill(data.phone)` — use first() since PhoneInput may render a hidden and visible input
6. Click "Próximo" button to advance to step 2
7. Wait 300ms for step transition
8. Click "Próximo" to advance to step 3
9. Wait 300ms for step transition
10. Click save button: `page.getByRole("button", { name: /cadastrar cliente|cadastrar/i })`
11. Wait for redirect: `page.waitForURL(/\/contacts$/, { timeout: 20000 })`

`editContact(contactId: string, data: { name?: string })`:
1. Navigate to `/contacts/${contactId}`
2. Wait for URL match (15s)
3. Wait for name input to be visible (15s)
4. If data.name: clear and fill name input
5. Click step button "Finalizar" to jump to step 3 (allowClickAhead=true for existing contacts)
6. Wait for save button, click it
7. Wait for URL to remain on `/contacts/${contactId}` or redirect (20s)
8. Wait 500ms to settle

`deleteContact(contactName: string)`:
1. Find the row: `page.locator("tr, [role='row']").filter({ has: page.getByRole("link", { name: contactName }) }).first()` — if no row found, fall back to `page.locator("div, li").filter({ hasText: contactName }).filter({ has: page.getByTitle("Excluir") }).last()`
2. Click delete button: `row.getByTitle("Excluir").click()`
3. Wait for AlertDialog title "Excluir Cliente" to be visible (8s)
4. Click confirm: `page.getByRole("button", { name: /^excluir$/i })`
5. Wait for contact link to be hidden: `page.getByRole("link", { name: contactName }).waitFor({ state: "hidden", timeout: 10000 })`

test("CONT-01: creates a contact via the UI form", async ({ authenticatedPage }) => {
  const contactsPage = new ContactsPage(authenticatedPage);
  const timestamp = Date.now();
  const contactName = `Contato Teste ${timestamp}`;

  await contactsPage.goto();
  await contactsPage.isLoaded();

  await contactsPage.createContact({ name: contactName, phone: "(11) 99999-0001" });

  // Verify the contact appears in the list
  await contactsPage.goto();
  await contactsPage.isLoaded();
  const contactLink = await contactsPage.getContactByName(contactName);
  await expect(contactLink).toBeVisible({ timeout: 10000 });

**Test 2 — CONT-02: Edit a contact**
```
test("CONT-02: edits a contact via the UI form", async ({ authenticatedPage }) => {
  const contactsPage = new ContactsPage(authenticatedPage);
  const timestamp = Date.now();
  const originalName = `Contato Editar ${timestamp}`;
  const updatedName = `Contato Editado ${timestamp}`;

  // Create via UI first
  await contactsPage.goto();
  await contactsPage.isLoaded();
  await contactsPage.createContact({ name: originalName, phone: "(11) 99999-0002" });

  // Navigate back to list and find the created contact to get its ID
  await contactsPage.goto();
  await contactsPage.isLoaded();

  // Get the link href to extract the ID
  const contactLink = await contactsPage.getContactByName(originalName);
  await expect(contactLink).toBeVisible({ timeout: 10000 });
  const href = await contactLink.getAttribute("href");
  expect(href).toBeTruthy();
  const contactId = href!.split("/contacts/")[1];
  expect(contactId).toBeTruthy();

  // Edit the contact
  await contactsPage.editContact(contactId, { name: updatedName });

  // Verify updated name appears in list
  await contactsPage.goto();
  await contactsPage.isLoaded();
  const updatedLink = await contactsPage.getContactByName(updatedName);
  await expect(updatedLink).toBeVisible({ timeout: 10000 });
});
```

**Test 3 — CONT-03: Delete a contact**
```
test("CONT-03: deletes a contact via the UI", async ({ authenticatedPage }) => {
  // Create via API in beforeEach pattern (inline here since we only need it for this test)
  const { idToken } = await signInWithEmailPassword(
    USER_ADMIN_ALPHA.email,
    USER_ADMIN_ALPHA.password,
  );

  const timestamp = Date.now();
  const contactName = `Contato Deletar ${timestamp}`;

  const createResponse = await authenticatedPage.request.post("/api/backend/v1/clients", {
    headers: { Authorization: `Bearer ${idToken}` },
    data: {
      name: contactName,
      phone: "(11) 99900-0003",
      types: ["cliente"],
    },
  });
  expect(createResponse.status()).toBe(201);

  const contactsPage = new ContactsPage(authenticatedPage);
  await contactsPage.goto();
  await contactsPage.isLoaded();

  // Verify contact appears in list before deleting
  await expect(authenticatedPage.getByRole("link", { name: contactName })).toBeVisible({ timeout: 10000 });

  await contactsPage.deleteContact(contactName);

  // Verify the contact is gone
  await expect(authenticatedPage.getByRole("link", { name: contactName })).not.toBeVisible({ timeout: 10000 });
});
```
