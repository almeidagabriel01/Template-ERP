# Plan 15-08: Free Plan UI Gating for LiaContainer

## Objective

Close the CHAT-08 gap by preventing free-plan users from seeing the Lia trigger button and panel in ProtectedAppShell, using `useAuth().user?.role !== "free"` as the gating mechanism.

## Context

Phase 15 verification found that `ProtectedAppShell` guards LiaContainer with `planTier !== undefined`, which passes for ALL logged-in users because `usePlanLimits` maps free-plan users to `planTier = "starter"` (the `PlanTier` type has no `"free"` value). The correct signal for free-plan detection is `useAuth().user?.role === "free"`, which is an established pattern used in `src/app/login/_hooks/useLoginForm.ts` and `src/app/login/page.tsx`. The `User` type in `src/types/index.ts` defines `role: "admin" | "user" | "superadmin" | "free" | "member"`.
