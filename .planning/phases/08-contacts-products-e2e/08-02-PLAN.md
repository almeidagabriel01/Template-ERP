### Task 1: Create ProductsPage POM (`e2e/pages/products.page.ts`)

Create a new file `e2e/pages/products.page.ts` with a class-based POM following the pattern of `e2e/pages/proposals.page.ts`.

The class must have:

**Constructor properties:**
- `page: Page`
- `newProductButton: Locator` — `page.getByRole("link", { name: /novo produto|novo cadastro|adicionar produto/i })`
- `pageHeading: Locator` — `page.locator('h1').first()`

**Methods:**

`goto()`: Navigate to `/products`.

`isLoaded()`: Wait for URL to match `/products$` (15s timeout), return true.

`getProductByName(name: string)`: Return `this.page.getByRole("link", { name }).first()` (product names render as links to /products/{id}).

`createProduct(data: { name: string; category: string; price: string })`:
1. Click `newProductButton`
2. Wait for URL `/products/new` (15s)
3. Wait for `#name` input to be visible (15s)
4. Fill name: `page.locator('#name').fill(data.name)`
5. **Category via DynamicSelect/SearchableSelect:**
   - Find the SearchableSelect input for category — it has `searchPlaceholder="Digite para buscar..."`. Use: `page.locator('input[placeholder*="buscar" i]').first()`
   - Click it to open the dropdown: `.click()`
   - Type category: `.fill(data.category)`
   - Wait for "Cadastrar" button to appear: `page.getByRole("button", { name: /cadastrar/i }).waitFor({ state: "visible", timeout: 5000 })`
   - Click the Cadastrar button: `page.getByRole("button", { name: /cadastrar/i }).first().click()`
   - Wait 500ms for option to be created and selected
6. Click "Próximo" to advance to step 2 (Preço)
7. **Price via CurrencyInput:**
   - Wait for `#price` to be visible (10s)
   - Click `#price` to focus it
   - Use `page.keyboard.type(data.price)` — pass "15000" for 150.00
8. Click "Próximo" to advance to step 3 (Imagens)
9. Wait 300ms
10. Click "Próximo" to skip images and advance to step 4 (Resumo)
11. Wait 300ms
12. Click submit button: `page.getByRole("button", { name: /salvar|criar produto|confirmar/i })`
13. Wait for redirect: `page.waitForURL(/\/products$/, { timeout: 20000 })`

`editProduct(productId: string, data: { name?: string })`:
1. Navigate to `/products/${productId}`
2. Wait for URL match (15s)
3. Wait for `#name` to be visible (15s)
4. If data.name: clear and fill name — `page.locator('#name').clear(); await page.locator('#name').fill(data.name)`
5. Click step button "Resumo" to jump to last step: `page.getByRole("button", { name: /resumo/i }).click()`
6. Wait for submit button visible (10s): `page.getByRole("button", { name: /salvar|criar produto|confirmar/i })`
7. Click submit
8. Wait 1000ms to settle

`deleteProduct(productName: string)`:
1. Find the row containing the product link: `page.locator("tr, [role='row']").filter({ has: page.getByRole("link", { name: productName }) }).first()` — fall back to `page.locator("div, tr").filter({ has: page.getByTitle("Excluir") }).filter({ hasText: productName }).last()`
2. Click delete button: `row.getByTitle("Excluir").click()`
3. Wait for AlertDialog title "Excluir Produto" (8s)
4. Click confirm: `page.getByRole("button", { name: /^excluir$/i })`
5. Wait for product link to be hidden: `page.getByRole("link", { name: productName }).waitFor({ state: "hidden", timeout: 10000 })`

```
test("PROD-01: creates a product via the UI form", async ({ authenticatedPage }) => {
  const productsPage = new ProductsPage(authenticatedPage);
  const timestamp = Date.now();
  const productName = `Produto Teste ${timestamp}`;

  await productsPage.goto();
  await productsPage.isLoaded();

  await productsPage.createProduct({
    name: productName,
    category: "Sensores",
    price: "15000", // 150.00 in cents format for CurrencyInput
  });

  // Verify the product appears in the list
  await productsPage.goto();
  await productsPage.isLoaded();
  const productLink = await productsPage.getProductByName(productName);
  await expect(productLink).toBeVisible({ timeout: 10000 });
});
```

**Test 2 — PROD-02: Edit a product**
```
test("PROD-02: edits a product via the UI form", async ({ authenticatedPage }) => {
  const productsPage = new ProductsPage(authenticatedPage);
  const timestamp = Date.now();
  const originalName = `Produto Editar ${timestamp}`;
  const updatedName = `Produto Editado ${timestamp}`;

  // Create via UI first
  await productsPage.goto();
  await productsPage.isLoaded();
  await productsPage.createProduct({
    name: originalName,
    category: "Sensores",
    price: "15000",
  });

  // Navigate back to list to find the created product's ID
  await productsPage.goto();
  await productsPage.isLoaded();

  const productLink = await productsPage.getProductByName(originalName);
  await expect(productLink).toBeVisible({ timeout: 10000 });
  const href = await productLink.getAttribute("href");
  expect(href).toBeTruthy();
  const productId = href!.split("/products/")[1];
  expect(productId).toBeTruthy();

  // Edit the product name
  await productsPage.editProduct(productId, { name: updatedName });

  // Verify updated name appears in list
  await productsPage.goto();
  await productsPage.isLoaded();
  const updatedLink = await productsPage.getProductByName(updatedName);
  await expect(updatedLink).toBeVisible({ timeout: 10000 });
});
```

**Test 3 — PROD-03: Delete a product**
```
test("PROD-03: deletes a product via the UI", async ({ authenticatedPage }) => {
  // Create via API to avoid using the seeded product-sensor-001 (used in proposals)
  const { idToken } = await signInWithEmailPassword(
    USER_ADMIN_ALPHA.email,
    USER_ADMIN_ALPHA.password,
  );

  const timestamp = Date.now();
  const productName = `Produto Deletar ${timestamp}`;

  const createResponse = await authenticatedPage.request.post("/api/backend/v1/products", {
    headers: { Authorization: `Bearer ${idToken}` },
    data: {
      name: productName,
      price: 100,
      category: "Teste",
      markup: 30,
      itemType: "product",
    },
  });
  expect(createResponse.status()).toBe(201);

  const productsPage = new ProductsPage(authenticatedPage);
  await productsPage.goto();
  await productsPage.isLoaded();

  // Verify product is in list
  await expect(authenticatedPage.getByRole("link", { name: productName })).toBeVisible({ timeout: 10000 });

  await productsPage.deleteProduct(productName);

  // Verify the product is gone
  await expect(authenticatedPage.getByRole("link", { name: productName })).not.toBeVisible({ timeout: 10000 });
});
```
