```typescript
interface TransactionCreateData {
  type: 'income' | 'expense';
  description: string;
  date?: string;       // YYYY-MM-DD, defaults to today via DatePicker "Hoje"
  amount: string;      // e.g. "1500.00" — CurrencyInput format

  notes?: string;      // Optional, step 3
}
```

**Add these methods to TransactionsPage:**

1. `async createTransaction(data: TransactionCreateData): Promise<void>`
   - Call `this.clickNewTransaction()`, wait for URL `/transactions/new`
   - **Step 0 (Type):** Click the card/button matching `data.type` — use `page.getByRole('button', { name: data.type === 'income' ? /receita/i : /despesa/i })`. Then click "Proximo".
   - **Step 1 (Details):** Fill description via `page.getByLabel(/descricao/i)` or `page.locator('#description')`. For date: click the DatePicker trigger button, click "Hoje" (today) in the calendar. Then click "Proximo".
   - **Step 2 (Payment):** Fill amount in CurrencyInput — locate via label or `[data-testid]`. For dueDate (if income): same DatePicker pattern as date. For wallet: click WalletSelect dropdown, wait for options to load (pitfall 2: `waitFor` visible state), select by wallet name text. Then click "Proximo".
   - **Step 3 (Review):** Optionally fill notes textarea. Click submit button (text: /salvar|confirmar|criar transacao/i). Wait for redirect to `/transactions` (list page).
   - IMPORTANT: Read the actual form component source to discover exact labels, IDs, and data-testid attributes. The skeleton above is directional — the real locators must come from reading the source files in read_first.

2. `async editTransaction(description: string, newData: Partial<TransactionCreateData>): Promise<void>`
   - Navigate to the transactions list via `this.goto()`, find the transaction row by description, click to open detail/edit view.
   - Once on the edit page (`/transactions/[id]`), the wizard re-renders with existing data.
   - Modify only the fields present in `newData` (e.g., clear and fill description if `newData.description` is provided).
   - Navigate to the final step and click save/submit. Wait for redirect confirmation.

3. `async deleteTransaction(description: string): Promise<void>`
   - Find the transaction row using `getTransactionByDescription(description)`.
   - Click the row's action menu (three-dot button or "Mais acoes" button — follow the proposals.page.ts `deleteProposal` row-targeting pattern).
   - Click "Excluir" in the dropdown menu.
   - Confirm in the AlertDialog by clicking the confirm "Excluir" button.
   - Wait for the transaction to disappear from the list.

4. `async getTransactionByDescription(description: string): Promise<Locator>`
   - Return a Locator for the transaction row/card containing the description text.
   - Use Playwright filter chains scoped to transaction item containers (pitfall 5: avoid `page.getByText` which matches multiple elements).
   - Pattern: `this.page.locator('[data-testid="transaction-item"], [data-testid="transactions-list"] > div').filter({ hasText: description }).first()` — adjust based on actual DOM.

**Test block 1 — FIN-01: Create transaction:**
```typescript
test.describe("FIN-01: Create transaction", () => {
  test("creates an income transaction via UI wizard and it appears in the list", async ({ authenticatedPage }) => {
    const transactionsPage = new TransactionsPage(authenticatedPage);
    await transactionsPage.goto();
    await transactionsPage.isLoaded();

    const testDescription = `Test Income ${Date.now()}`;

    await transactionsPage.createTransaction({
      type: 'income',
      description: testDescription,
      amount: '1500.00',
      walletName: 'Conta Principal',
    });

    // Navigate back to transactions list
    await transactionsPage.goto();
    await transactionsPage.isLoaded();

    // Assert the new transaction appears
    const item = await transactionsPage.getTransactionByDescription(testDescription);
    await expect(item).toBeVisible();

    // Cleanup: delete the created transaction (create-then-delete per D-03)
    await transactionsPage.deleteTransaction(testDescription);
  });
});
```
